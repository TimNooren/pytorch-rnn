
### PyTorch Character Based RNN

A recurrent neural network that reads "The Adventures of Sherlock Holmes" ([link](http://www.gutenberg.org/ebooks/1661)), and generates sentences of its own as it trains.

To start training:
```bash
python -m char_rnn.main
```
